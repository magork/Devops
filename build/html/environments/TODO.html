Questions

  1. What is not a cloud computing service?

    1. Infrastructure as a Service (IaaS) IaaS is a pay-as-you-go cloud computing service that offers on-demand storage and networking resources.

    2. Software as a Service (SaaS) This service allows clients and customers to connect with cloud-based apps that provide email, teleconferencing, productivity, and storage services.

    3. Code as a Service CODEaaS offers you complete, no-to-low code integration with your software.

    4. Platform as a Service (PaaS) PaaS offers complete cloud-based development and deployment where a client can create simple or the most sophisticated cloud-based applications.

    5. Serverless Serverless cloud computing is an architecture where the service provider manages the execution of the code instead of deploying them on servers.”*

  2. What is not a model of deployment in the cloud?

    1. Private Cloud

    2. Public Cloud

    3. Deep Web/Dark Cloud

    4. Community Cloud

    5. Hybrid Cloud

  3. What is not an advantage of cloud computing?

    1. accessibility

    2. cost-effective

    3. security

    4. performance

    5. flexibility

    6. scalability

    7. usability
